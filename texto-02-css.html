Este texto es un ejemplo de texto preformado.

La marca pre preserva espacios en blanco y saltos de línea.

La marca pre también es buena para mostrar código de programación:

        for i = to 10
            print i
        next i

También se puede usar la marca code:.

        for i = 1 to 10 print i next 1

pero code no respeta espaciado ni saltos de línea.
